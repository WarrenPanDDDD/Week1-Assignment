# Week 1 Assignment

To get latest BTC/USD price from Cronos Oracle.

## Description

Get latest BTC/USD price from Cronos Oracle, log out in console every minute.

## Getting Started

### Dependencies

* nodejs
* ethers
* dotenv

### Installing

```
npm install
```

### Executing program

```
cd app

node read-btc-price.js 
```
